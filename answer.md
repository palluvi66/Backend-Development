1. Presentation Layer
2. Application (Business) Layer
3. Data Layer

1. Presentation Layer
This is the user interface layer.
It includes:Web pages,Forms, buttons, menus,All visual components
Responsibilities of this layer are to display data to the user,Collect user input,Send requests to the application layer

2. Application (Business) Layer
This is the core logic layer of the application.
Responsibilities of this layer are to process user requests,Apply business rules, perform calculations and validations,Communicate between presentation and data layer

Technologies used in this layer are
Node.js, Java, Python, PHP, .NET, etc.

3. Data Layer
This layer handles data storage and management.
Responsibilities of this layer are to store and retrieve data,Manage database operations (CRUD),Ensure data integrity and security
